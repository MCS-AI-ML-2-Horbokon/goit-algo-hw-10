## Обчислення визначеного інтеграла методом Монте-Карло

Виконано обчислення визначеного інтеграла функції `f(x) = sin(x) * x` на інтервалі [0, 3] двома методами:

1. **Метод quad (SciPy)**: 3.1111 ± 3.45×10⁻¹⁴ (еталонне значення)

2. **Метод Монте-Карло**:
   - 10,000 точок: 3.1095 (похибка 0.05%)
   - 100,000 точок: 3.1229 (похибка 0.38%)
   - 1,000,000 точок: 3.1067 (похибка 0.14%)
   - 10,000,000 точок: 3.1125 (похибка 0.04%)

**Висновок**: Метод Монте-Карло показує хорошу збіжність до аналітичного результату. Зі збільшенням кількості точок точність зростає. Для досягнення похибки < 0.1% потрібно мінімум 10 мільйонів точок. Метод підходить для складних функцій, де аналітичне інтегрування неможливе, для швидкого обчислення приблизного значення.
